# PicoAreaHelper
Pico4换区小助手

![image](https://user-images.githubusercontent.com/51113234/195987643-9224c1a6-76f3-42c1-aa72-679cc825ad34.png)


## 功能列表
- 打开区域设置
- 切换成中国
- 切换成海外
- 重启设备

## 使用方法

### 切换到海外
1. 打开区域设置，切换国家代码为JP
2. 输入2切换成海外
3. 完成

### 切换成国区
1. 打开区域设置，切换国家代码为CN
2. 输入1切换成中国
3. 输入3重启设备
4. 完成
