# PicoAreaHelper
Pico4换区小助手

![image](https://user-images.githubusercontent.com/51113234/195965014-0bcade21-3d44-4a23-bdbb-31ea507af45f.png)

## 功能列表
- 打开区域设置
- 切换成国区
- 切换成海外
- 重启设备

## 使用方法

### 切换到海外
1.打开区域设置，切换国家代码为JP
2.输入2切换成海外
3.完成

### 切换成国区
1.打开区域设置，切换国家代码为CN
2.输入1切换成海外
3.输入3重启设备
4.完成
